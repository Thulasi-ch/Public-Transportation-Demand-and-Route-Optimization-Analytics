# Public Transportation Demand and Route Optimization Analytics

## 📌 Project Overview
This project analyzes public transportation data to predict passenger demand, optimize routes, and improve transport efficiency using Data Analytics and Machine Learning.

## 🎯 Objectives
- Predict passenger demand
- Identify peak travel hours
- Optimize transport routes
- Recommend bus allocation
- Visualize insights using dashboard

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Streamlit

## 📂 Project Structure
code/ - Python source files  
dataset/ - Input datasets  
charts/ - Output graphs  
output/ - Result CSV files  
documents/ - Report and PPT

## 📈 Features
- Demand Forecasting
- Route Optimization
- Peak Hour Analysis
- Congestion Analysis
- Interactive Dashboard

## ▶️ How to Run

pip install -r requirements.txt
streamlit run code/dashboard.py

📊 Results
Improved route planning

Better bus allocation

Reduced congestion

Easy data visualization
